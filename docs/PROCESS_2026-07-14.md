## **给您的 1 步行动(早上起来请做)**

### 在 Supabase 网页取 DB 连接串(2 分钟)

1. 浏览器打开 https://supabase.com/dashboard → 进您那个 `fxs-84's Project`
2. 左侧 **Settings**(齿轮) → **Database**
3. 滚到 **Connection string** 那段,选最上面的 **URI** 这一行
4. 整段复制(长得像 `postgres://postgres.xxxxx:PASSWORD@aws-X-region.pooler.supabase.com:6543/postgres`)

> **重要 — 选对端口**:
> - `5432`(Direct connection)— DDL 用,**选这个**
> - `6543`(Transaction pooler)— 给客户端用,DDL 有时跑不掉
> - 如果您看到 "Connection pooling mode" 切换,**选 Session** 那行复制

### 把它粘进 `.env.local`(30 秒)

```
notepad app/.env.local
```

在文件末尾加一行:

```
SUPABASE_DB_URL=postgres://postgres.您的ref:密码@aws-0-地区.pooler.supabase.com:6543/postgres
```

**Ctrl+S 保存,关掉。**

### 跑一行命令(30 秒,跑完就完事)

cd app
npm run setup:supabase
